The first step to getting PhoneGap up and running is proper installation. Follow the instructions that can be found here: Installation instructions. At this point you should have an application running on your device, I am using an Android device but this should hold true for any device that is supported.

Let's look at two directories in particular that are generated by the tool:

There are a few files generated by the tool that a worth a closer look:
src

Contained in this directory is the startup source file for PhoneGap. Written in Java, you can modify the code in this file to access the java layer used by PhoneGap. This file can also contain code that loads custom Java code that your script can use. More on this later as well as an example.
